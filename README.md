# Google Earth Engine script for Ecosphere Fig. 1

This repository contains the Google Earth Engine JavaScript code used to identify and visualize Landsat imagery for Fig. 1 in:

Trout-Haney et al. Accepted . Key considerations for the use of satellite-based remote sensing for lake and reservoir research and management. _Ecosphere_.

The script uses Landsat Collection 2 Tier 1 Level 2 surface reflectance imagery from Google Earth Engine.

## Contents

- `fig1_gee_landsat_script.js`: Google Earth Engine JavaScript code used for Fig. 1.
